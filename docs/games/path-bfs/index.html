BFS: shift() vs ring queue

Full grid flood fill (4-neighbor BFS) from the center on a 56×56 grid, recomputed every frame. TypeScript uses an Array with shift() for the FIFO — each dequeue moves every remaining element. Selvr-style uses a fixed Int32Array queue with head/tail indices (ring buffer): O(1) enqueue and dequeue.

Why this is fair

  • Same traversal order and visited semantics; only queue implementation and visited storage differ.
  • Production TS code often uses index-based queues; this demo intentionally uses shift() to show the cost of a common pitfall.
