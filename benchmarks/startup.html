How this is measured

  • Parse time: measured by calling new Function(src + "/*N*/") where N increments each iteration. The unique suffix prevents V8 from reusing a previously compiled function from its source cache. Elapsed time is totalled over all iterations and divided to get a stable per-parse average.
  • Iteration count: auto-scaled by bundle size — 500+ iterations for small bundles (Selvr, vanilla JS) down to 10 for large bundles (Angular, Next.js) — so total measurement time stays manageable without sacrificing accuracy.
  • Bundle content: Selvr and vanilla JS use actual code (the real selvr build output, 2918 bytes). All framework bundles are synthetic — same size as the real frameworks but built from realistic CommonJS/ESM patterns (closures, prototypes, class transforms) to match real parse load.
  • What this measures: JS parse and compile time only — not network transfer. Selvr's smaller bundles also benefit from faster download, which compounds the advantage at scale.
